# Medap

## Visão Geral

O Medap é um sistema desenvolvido como projeto de TCC. Seu objetivo consiste em fornecer um ambiente voltado para o gerenciamento de estabelecimentos e profissionais da área da saúde, assim como a emissão de receitas.

A plataforma permite que os usuários administrem diferentes aspectos, como estabelecimentos, cadastro de múltiplas unidades, gerenciamento de profissionais, medicamentos, pacientes e criação de prescrições.
